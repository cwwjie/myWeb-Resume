我穿越我穿越无数世界线只为守护与你的约定

2016年8月31日

苦恼着，歇斯底里着，痛苦着，不断挣扎的数月时间，这一切会在未来的某一瞬间得到回报。我们或许就是被那个瞬间迷住的一种无可救药的生物吧。


我曾经问过她为什么要对非亲非故的我这么好，然后那个人笑着这么说了，时光流转，人总是在他人的帮助下生活着，所以总有一天你也要帮助别人啊。一步一步地踏出去，鞭策着自己，互相激励，互相扶持，不断前进只有我止步不前，是时间停止了吧！因为你喜欢她,所以你的眼中就只有她了。


飞机穿梭于茫茫星海中逐渐远去，你如猫般，无声靠近，从意想不到的角度玩弄他人，而我只能呆愣在原地，永远只能跟随你的步伐。
